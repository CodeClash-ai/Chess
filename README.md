# CodeClash Chess Arena

This repository contains the chess engine and match runner for the CodeClash chess arena.

## Structure

- `kojiro/` - The Kojiro UCI-compatible chess engine (source: [Babak-SSH/Kojiro](https://github.com/Babak-SSH/Kojiro))
- `fastchess/` - The Fastchess match runner (source: [Disservin/fastchess](https://github.com/Disservin/fastchess))

For more information about CodeClash, see [CodeClash-ai/CodeClash](https://github.com/CodeClash-ai/CodeClash).
